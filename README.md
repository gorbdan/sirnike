# Сырник — AI бот генерации

## Запуск

```powershell
cd "C:\Users\Аня\OneDrive\Desktop\telegram bots\ai bot"
py sirnike.py
```

## Основные команды

- `/start` — главное меню.
- `/balance` — баланс и бесплатные генерации.
- `/buy` — покупка изюминок.
- `/ref` — реферальная ссылка.
- `/ai <вопрос>` — текстовый AI-ассистент через MashaGPT Chat.

## Админ-команды

- `/admin_add <user_id> <amount>` — начислить изюминки.
- `/broadcast_promo <prompt>` — промо-рассылка по фото в реплае + кнопка «Попробовать».
- `/broadcast_text <текст>` — текстовая рассылка без фото и кнопок.
- `/promo_stats <promo_id>` — статистика по промо-рассылке.
- `/audience_stats [days]` — портрет аудитории и метрики.

## Библиотека промтов (все команды)

- `/pl_admin` — кнопочный админ-редактор библиотеки.
- `/pl_list` — список категорий и количество шаблонов.
- `/pl_newcat <название>` — создать категорию.
- `/pl_renamecat <старое> | <новое>` — переименовать категорию.
- `/pl_delcat <название>` — удалить категорию.
- `/pl_export` — выгрузить текущий `prompt_library.json`.
- `/pl_history` — история генераций (выбор и экспорт в библиотеку).
- `/pl_save [название]` — сохранить последнюю генерацию в библиотеку.
- `/pl_import <название> | <промпт>` — импорт старой генерации из реплая на фото.

## Как сохранить промт в библиотеку (все способы)

1. Последняя генерация:
- Сделай генерацию.
- Введи `/pl_save` или `/pl_save <название>`.
- Выбери категорию кнопкой.
- Бот сохранит и картинку-пример, и промпт.

2. Из истории генераций:
- Открой `/pl_history` (или через `/pl_admin` → «История генераций»).
- Выбери нужную запись.
- Нажми «Сохранить в библиотеку ✅».
- Выбери категорию.

3. Из старого фото (которое уже есть в чате):
- Ответь (reply) на фото командой:
- `/pl_import <название> | <промпт>`
- После этого шаблон появится в библиотеке с этим промптом и фото-примером.

Важно:
- Для качественного шаблона указывай полноценный промпт.
- При нажатии «Использовать промпт ✅» в библиотеке промпт подставляется в буфер автоматически.

## WebApp библиотеки

В `.env`:

```text
PROMPT_WEBAPP_URL=https://your-site.netlify.app
```

Важно:
- Бот синхронно обновляет оба файла:
  - `webapp/prompt_library.json`
  - `prompt_library.json`
- Если WebApp на Netlify, после правок перезаливай папку `webapp` в Netlify Drop.

## Чтобы обновления библиотеки точно отобразились

После изменения `prompt_library.json` сделай это по шагам:

1. Сохрани изменения в `prompt_library.json` (в корне бота).
2. Если используешь WebApp: скопируй обновленный файл в `webapp/prompt_library.json`.
3. Перезалей папку `webapp` на Netlify (если WebApp на Netlify).
4. Перезапусти бота (`py sirnike.py`).
5. В Telegram закрой и заново открой библиотеку промтов (иногда помогает закрыть чат и открыть снова).

Если видишь «старую» библиотеку:
- Проверь, что изменился именно тот файл, который реально используется.
- Убедись, что на Netlify задеплоена новая версия (не кэш старой).
